## What on Earth is This Repo About?

This virtual carnival is designed to aid our fellow Chinese redditors in reuniting with their beloved subreddits. Like a master magician, we bring back the lost worlds of Reddit!

## Repo Contents

- **txt_links.txt**: A comprehensive list of all Chinese-related subreddit links.
- **links.md**: A Markdown file for easy viewing and access to subreddit links. If you are comfortable with code, please view this file directly. üòä
- **open.py**: The heart of our project! This script assists in revitalizing your Reddit experience. Use it to open a range of subreddit links directly from the command line.
- **README.md, README_CN.md, README_JP.md**: Instruction guides in English, Chinese, and Japanese.

## üöÄ How to Join the Circus üöÄ

### Step 1: Clone This Repo

```bash
git clone https://github.com/kanagawasurfin/home-of-shushu.git
```

### Step 2: Enter the Magical World

```bash
cd home-of-shushu
```

### Step 3: Run the `open.py` Script

```bash
python3 open.py
```

Follow the on-screen prompts to open a range of subreddit links directly in your browser.

## Using the `open.py` Script

1. When prompted, enter the range of lines you wish to open (e.g., `4,10` to open subreddits from line 4 to 10 in `txt_links.txt`).
2. Review the subreddit names that are about to be opened.
3. Confirm with `y` to proceed or `n` to cancel.
